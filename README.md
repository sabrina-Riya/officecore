# OfficeCore

## Description
A leave management system built with Node.js, Express, Passport, and PostgreSQL.

## Features
- Employee dashboard
- Admin dashboard
- Role-based access control (RBAC)
- Leave management (apply, approve, reject)
- User authentication with Passport
- Flash messages for feedback

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- EJS for templating
- Passport.js for authentication
- bcrypt for password hashing
- express-session and express-flash
